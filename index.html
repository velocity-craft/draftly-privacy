Draftly 개인정보처리방침

시행일: 2025년 2월 18일

Draftly(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」에 따라 아래와 같이 개인정보처리방침을 수립하여 공개합니다.

1. 수집하는 개인정보 항목

앱은 서비스 제공을 위해 다음의 정보를 수집할 수 있습니다.

  • 음성 데이터: 음성 입력 기능 사용 시 마이크를 통해 녹음된 음성 (RECORD_AUDIO 권한)
  • 기기 정보: 광고 식별자, 기기 모델, OS 버전 (광고 서비스 제공 목적)

앱에서 작성하는 메모, 체크리스트 등의 콘텐츠는 사용자의 기기에만 저장되며 외부 서버로 전송되지 않습니다.

2. 개인정보의 수집 및 이용 목적

  • 음성 입력 기능: 사용자의 음성을 텍스트로 변환하여 메모 작성을 지원합니다.
  • 광고 제공: Google AdMob을 통한 맞춤형 광고를 표시합니다.
  • 앱 개선: 비정상 종료 분석 및 서비스 안정성 향상을 위해 익명화된 사용 데이터를 수집할 수 있습니다.

3. 개인정보의 보유 및 이용 기간

  • 음성 데이터: 텍스트 변환 즉시 삭제되며 별도로 저장하지 않습니다.
  • 기기 내 데이터: 앱 삭제 시 모든 데이터가 함께 삭제됩니다.
  • 광고 관련 정보: Google의 개인정보처리방침에 따릅니다.

4. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 원칙적으로 제3자에게 제공하지 않습니다. 다만, 아래의 경우는 예외로 합니다.

  • 사용자가 사전에 동의한 경우
  • 법령에 의해 요구되는 경우
  • 광고 서비스 제공을 위해 Google AdMob에 제한적 정보가 공유될 수 있습니다.

5. 앱 접근 권한

앱은 다음의 접근 권한을 사용합니다.

  • 마이크 (RECORD_AUDIO): 음성 입력을 통한 메모 작성 [선택]
  • 인터넷 (INTERNET): 광고 표시 및 음성 인식 API 연동 [필수]
  • 알림 (POST_NOTIFICATIONS): 리마인더 알림 전송 [선택]
  • 다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW): 플로팅 빠른 메모 버튼 [선택]
  • 포그라운드 서비스 (FOREGROUND_SERVICE): 플로팅 버튼 백그라운드 유지 [필수]
  • 부팅 시 실행 (RECEIVE_BOOT_COMPLETED): 리마인더 알림 재등록 [필수]

선택 권한은 해당 기능 사용 시에만 요청되며, 허용하지 않아도 앱의 기본 기능을 사용할 수 있습니다.

6. 개인정보의 안전성 확보 조치

앱은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

  • 모든 사용자 데이터는 기기 내부 저장소에 암호화하여 저장합니다.
  • 네트워크 통신 시 암호화된 연결(HTTPS)을 사용합니다.
